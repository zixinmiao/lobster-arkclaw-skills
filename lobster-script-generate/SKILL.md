# 导购话术建议

根据客户状态生成可直接使用的话术。

## 输入关注点
优先识别：
- 客户当前状态
- 最近互动节点
- 商品与试穿反馈
- 异议类型
- 目标动作（到店、加购、成交、回访、复购）

## 输出目标
输出 `result`：
- `customer_state`
- `goal`
- `next_action`
- `script`
- `alternate_scripts`
- `risk_notes`

## 生成规则
- 话术必须像一线导购能直接发出的内容。
- 语气积极，但不过度推销。
- 必须体现下一步动作。
- 如果存在负面反馈或明确异议，要给处理口径，不回避。
- 避免空泛表达，如“多关注一下”“再跟进下”。

## 输出格式
返回：
```json
{"result": {...}}
```
